  • Class Summary 
    Class Description
    MuleMavenPluginVersionRule This rule ensures that the `mule-maven-plugin` of a specified version exists in the `pom.xml`.
    MuleRuntimeVersionRule This rule ensures that the `app.runtime` property within the `pom.xml` matches a given value.
    MunitMavenPluginAttributesRule This rule ensures that the `munit-maven-plugin` exists and is configured correctly.
    MunitPluginVersionRule This rule checks that the version number for the `munit-maven-version` plugin matches a given value.
    MunitVersionRule This rule checks that the version number for the `munit` plugin matches a given value.
    PomDependencyVersionRule This rule checks for the presence of a given dependency with a given version in the `pom.xml`.
    PomExistsRule This rule checks for the existence of a `pom.xml` file at the root of the project.
    PomPluginAttributeRule This rule checks for the existence of a given set of attributes in a given maven plugin within the `pom.xml`.
    PomPropertyValueRule This rule checks for the existence of a property with a given value in the `pom.xml`.
