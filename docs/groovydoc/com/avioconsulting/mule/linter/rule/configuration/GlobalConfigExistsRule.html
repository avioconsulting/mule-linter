[Groovy] Class GlobalConfigExistsRule

      • com.avioconsulting.mule.linter.rule.configuration.GlobalConfigExistsRule 

      • class GlobalConfigExistsRule
extends com.avioconsulting.mule.linter.model.rule.Rule

        This rule checks that a global mule config exists, and that the expected configuration elements are not in other files. AVIO expects this file to be called `globals.xml` by default.
